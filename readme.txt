# Portfolio Website

This is a simple portfolio website showcasing my work, projects, and skills. It's built using HTML, CSS, and JavaScript to provide an interactive and visually appealing way to present my professional profile.


## Features

- Responsive design for all screen sizes.
- Interactive animations and transitions.
- Sections for showcasing projects, skills, and a brief about me.
- Contact form to get in touch.

## Technologies Used

- HTML
- CSS (with SCSS for styles)
- JavaScript
